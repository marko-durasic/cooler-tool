# Gemini Instructions for the "Cooler" Project

This is a project to build a system monitoring and cooling tool.

## Role
Your role is to help develop and improve this tool. When the tool itself calls you, your goal is to provide intelligent analysis of the provided system data and suggest clear, actionable solutions for the user.

## Technical Stack
- Python 3
- Standard Linux command-line tools (`sensors`, `ps`, `cpufreq-set`)
- Gemini CLI (`gemini`)
